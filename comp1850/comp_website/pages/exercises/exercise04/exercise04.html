    • What to learn

      In this tutorial, I learned to create buttons in Photoshop applying patterns and save them as sprites to use on web pages. The first step that I learned was to create the button form as Shape layer in Photoshop. In this way, it is easier to change the shape properties and make selections over it. Then I learned the proper way to create and apply patterns in Photoshop. On the buttons created I used a pattern created by myself (rectangle button). This pattern is composed of diagonal stripes. In the second button (round one) I used a pattern from the Photoshop. I also learned how to organize the layers in folders to ease make changes on the characteristics of the button, either on normal and hover states.
