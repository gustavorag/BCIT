CSS Styles can be applyed to html tags, ids and classes in many different ways. Not only directly to the element but also combine syntaxes to apply styles to descendants elements. For instance, suppose that you want to create a unsorted list (<ul>) and want to change the style of all itens (descendants) inside this list. What we want to do is to make all itens rounded, remove de underline and change the color of the <a> tag and add some icons.

  1. We can do this by creating classes and applying them to respective tags:
    <ul>
        <li class="my_listitem">
          <a href="#" class="my_listitem_anchor">
            <span class="my_listitem_span">&#9745;</span>
              Button1
          </a>
        </li>
        <li class="my_listitem">
          <a href="#" class="my_listitem_anchor">
            <span class="my_listitem_span">&#9982;</span>
              Button2
          </a>
        </li>
        <li class="my_listitem">
          <a href="#" class="my_listitem_anchor">
            <span class="my_listitem_span">&#9919;</span>
              Button3
          </a>
        </li>
    </ul>

Or we can use the css capability to apply styles to descendants elements of a tag that has a specific class. To do this we should use the following notation: .class tagname

  1. Defining a class that affect descendants tags.
    .rounded_menu li{
        list-style-type: none;
        margin-left: 5px;
        margin-right: 5px;
        border: 2px solid #DDDDDD;
        border-radius: 50px;
        width: 100px;
        height: 100px;
        float: left;
        text-align: center;
        background-color: #EEEEEE;
        color: #BBBBBB;
        cursor:pointer;
    }

It means that every <li> tag inside an element that has the "rounded_menu" class will have these defined styles. The syntax ".className tagName" can be also combined with pseudo class and other descendants tags. Take a look on this other example:

  1. .rounded_menu li:hover{
        border: 2px solid #555555;
        color: #555555;
    }
  2. .rounded_menu li:hover a{
        color: #555555;
    }
  3. .rounded_menu li:hover span{
        transform: scale(1.5);
    }

Realize that the "hover" condition is triggered when the mouse is over the <li> tag (li:hover) but for each of these css definitions the resulting style is applied in different ways. The style 1 will be put on <li> tags inside the elemente with de "rounded_menu" class but the styles 2 and 3 will be applied to the <a> tags and <span> tags, respectivily, that are inside this especifically <li> tag. We can interprete the notation, ".rounded_menu li:hover span" in this way: For a <li> tag inside an element with class .rounded_menu and having the mouse over itself, apply the following styles to every <span> descendants tag of this <li> tag. In this way, you don't need to apply classes to every item inside the list. You can see the results next.


  1. CSS Definitions
    .rounded_menu{
        height: 100px;
        text-align: center;
    }
    .rounded_menu li{
        list-style-type: none;
        margin-left: 5px;
        margin-right: 5px;
        border: 2px solid #DDDDDD;
        border-radius: 50px;
        width: 100px;
        height: 100px;
        float: left;
        text-align: center;
        background-color: #EEEEEE;
        color: #BBBBBB;
        cursor:pointer;
    }
    .rounded_menu span{
        display: block;
        width: 100px;
        height: 60px;
        line-height: 60px;
        font-size: 1.5em; }

    .rounded_menu a{
        text-decoration: none;
        color: #BBBBBB;
    }

    .rounded_menu li:hover{
        border: 2px solid #555555;
        color: #555555;
    }

    .rounded_menu li:hover a{
        color: #555555;
    }
    .rounded_menu li:hover span{
        transform: scale(1.5);
    }
  2. HTML definitions

    <ul class="rounded_menu">
        <li><a href="#"><span>&#9745;</span>Button1</a></li>
        <li><a href="#"><span>&#9982;</span>Button2</a></li>
        <li><a href="#"><span>&#9919;</span>Button3</a></li>
    </ul>
