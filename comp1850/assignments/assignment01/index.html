  1. The <map> & <area> tags

    The <map> and <area> tags is used to create areas in a especific image. This areas can be used as normal links, like any <a> tag. The folloing code example shows how to use these both tags to create multiple link areas on one single image

    <img src="images/html-css-links.png" alt="HTML5 and CSS3 Logos" usemap="#imagemap">

    <map name="imagemap">
        <area shape="rect" coords="20,48,145,185"
          alt="Html5" href="https://www.w3schools.com/html/html5_intro.asp" target="_blank">
        <area shape="rect" coords="180,48,300,185"
          alt="Css3" href="https://www.w3schools.com/css/" target="_blank">
    </map>

    Click on HTML or CSS shilds to open a new page to W3C page

    Note that you need to reference the name of the map ("imagemap") in the attribute "usemap" inside the img tag, preciding by #. Inside the area tag you must inform the shape of the area and the coordinates following these rules:

    shapecoords
    For value "rect""left border, top border, right border, bottom border"
    For value "circ""centerx, centery, radius"
    For value "poly""x1, y1, x2, y2,..,xn, yn"

    !TIP: you can use a graphic design software, such as Adobe Photoshop, to identify the desired coordinates inside your image.
